ADDENDA GENERATOR

C++ developed billing invoice attachment.

The Addenda Generator was another small - big project that I was able to develop back when I used to work at Lumik Design.

Its main purpose was to pretty much save time. As part of Lumik, we were providers for Home Depot, and as such, Home Depot required certain modifications to the XML file generated by a regular invoice, in order to get paid.

Lumik Design until that moment was using third-party software to generate such complements for the XML file, however, it took about 20-30 minutes to complete just 1 complement, multiplied by 100, that's quite some time.

So, I put my hands to work and developed the Addenda Generator, using C++. This reduced the amount of time invested to a maximum of 1 minute per XML, plus it adds a simple UI which is more visual thant the previous app.
